
## Меню

### &lt;page&gt;

Ссылка на страницу

**Атрибуты**

|Наименование|Тип      |Описание          |Значение по умолчанию|
|------------|---------|------------------|---------------------|
|page-id     |Ссылка   |Ссылка на страницу|                     |
|route       |Строка   |URL страницы      |                     |
|label       |Строка   |Заголовок кнопки  |                     |
|icon        |CSS класс|Иконка            |                     |

**Пример**

```xml
<page page-id="ProtoPage" route="/proto" label="Контакты"/>
```

### &lt;a&gt;

Открыть страницу браузера

**Атрибуты**

|Наименование|Тип            |Описание                                                                       |Значение по умолчанию|
|------------|---------------|-------------------------------------------------------------------------------|---------------------|
|label       |Строка         |Подпись кнопки                                                                 |                     |
|href        |Строка         |Адрес URL                                                                      |                     |
|icon        |CSS класс      |Иконка                                                                         |                     |
|target      |self, newWindow|Сценарий открытия ссылки: self - в текущей вкладке, newWindow - в новой вкладке|self                 |

**Пример**

```xml
<a label="N2O" href="https://n2oapp.net/demo/"/>
```

### &lt;sub-menu&gt;

Кнопка с выпадающим меню

**Атрибуты**

|Наименование|Тип      |Описание      |Значение по умолчанию|
|------------|---------|--------------|---------------------|
|label       |Строка   |Подпись кнопки|                     |
|icon        |CSS класс|Иконка        |                     |

**Тело**

Список ссылок на [страницу](#$$__page_2$$), [внешние ссылки.](#$$__a_2$$)
или вложенное выпадающее меню

**Пример**

```xml
<sub-menu label="Ссылки" >
    <page page-id="ProtoPage" route="/proto" label="Контакты"/>
    <a label="N2O" href="https://n2oapp.net/demo/"/>
    <sub-menu>...</sub-menu>
</sub-menu>
```
